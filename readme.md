## imcheat : BEPINEX cheat plugin for Ultimate Zombie Defence 1
This cheat plugin is perfect for playing Ultimate Zombie Defence like a sandbox game, fully compatible with multiplayer.

**Features**

 - Godmode
 - Infinite damage
 - Wave skipper
 - Infinite money
 - Speedhack
 - Infinite bullets
 - Infinite build count limit
 - Disable building block
 - Halloween force

   These features **only work if you are the game host**, or **if you and the host** have the plugin ; exeptions are **speedhack** and **infinite bullets** wich work at any time. It is recommanded for all players in the lobby to have the plugin installed for smoother experience.

## How to install :
Just like any bepinex plugin installation :

 1. Find the game root folder. For Steam you can do : 
   right click (on the game)>manage>browse local files
 2. Download and extract Bepinex in the game root folder.
 3. Download the plugin and extract the zip in Bepinex/Plugins/.
 4. Download and extract the ConfigurationManager plugin to easily configure the cheat.
 5. Done ! You can launch the game. 
 6. Press **F1 to open the configuration menu at any time** ; please note that some cheats modifications like custom wave skip require restarting the game to take effect.
