## imcheat : BepInEx cheat plugin for Ultimate Zombie Defense 1
This cheat plugin does things that no other cheats do. It is perfect for playing Ultimate Zombie Defense like a sandbox game, fully compatible with multiplayer.

**🔧 Features :**

 - Godmode
 - Infinite bullet damage
 - Wave skipper
 - Infinite money
 - Speedhack
 - Infinite ammo
 - Infinite build count limit
 - Disable building block
 - force Halloween exclusive content

  🚨 These features **only work if you are the game host** and have the plugin, or **if you and the host** have the plugin ; exeptions are **speedhack** and **infinite bullets** which should always work. It is recommanded for all players in the lobby to have the plugin installed for smoother experience.
  
  **🏗 Planned features :**
 - Force build mode (build anytime)
 - Custom weapons fire rate

## 🚀 How to install :
Just like any bepinex plugin installation :

 1. Find the game root folder. For Steam you can do : 
   right click (on the game)>manage>browse local files
 2. Download and extract [BepInEx](https://github.com/BepInEx/BepInEx/releases) in the game root folder.
 3. Download [the plugin](https://github.com/immblind/ultimate-zombie-defense-bepinex-cheat/releases/) and extract the zip in BepInEx/plugins/. (should look like BepInEx/plugins/imcheat.dll)
 4. Download and extract the [ConfigurationManager](https://github.com/BepInEx/BepInEx.ConfigurationManager/releases) plugin for BepInEx to easily configure the cheat.
 5. Done ! You can launch the game. 
 6. Press **F1 to open the configuration menu at any time** ; please note that some cheats modifications like custom wave skip require restarting the game to take effect.

 🚧 The plugin may receive updates with additional features. Feel free to suggest some !
